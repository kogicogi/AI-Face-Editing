#### Progress Summary
>Five steps<br>
1. Prepare your own dataset or use [flickr-face-HQ Dataset](https://github.com/NVlabs/ffhq-dataset).
2. Preprocess your data.
3. Train using StyleGAN.
4. Edit your picture.
5. Restore picture resolution.
